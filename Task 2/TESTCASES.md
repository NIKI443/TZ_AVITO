### Тест-кейсы для проверки API микросервиса

**1. Создание объявления (POST /api/1/item)**

**Кейс создания:**

- **Запрос:** `POST /api/1/item`
- **Тело запроса:**
  ```json
  {
  	"sellerID": 800000,
  	"name": "test-advertisement",
  	"price": 2000,
  	"statistics": {
  		"likes": 10,
  		"viewCount": 230,
  		"contacts": 25
  	}
  }
  ```
- **Ожидаемый результат:** `200 OK`

**Проверка ввода неподходящих типов:**

- **Кейс 1:**

  ```json
  {
  	"sellerID": 800000,
  	"name": false,
  	"price": 2000,
  	"statistics": {
  		"likes": 10,
  		"viewCount": 230,
  		"contacts": 25
  	}
  }
  ```

  **Ожидаемый результат:** `400 Bad Request`

- **Кейс 2:**
  ```json
  {
  	"sellerID": 800000,
  	"name": "test-advertisement",
  	"price": "price",
  	"statistics": {
  		"likes": 10,
  		"viewCount": 230,
  		"contacts": 25
  	}
  }
  ```
  **Ожидаемый результат:** `400 Bad Request`

**2. Получение объявления по ID (GET /api/1/item/{id})**

| **Сценарий**               | **Ожидаемый результат**                                |
| -------------------------- | ------------------------------------------------------ |
| Действительный ID          | `200 OK` — возвращается объявление с указанным ID      |
| ID несуществующего объекта | `404 Not Found` — объявление с указанным ID не найдено |
| Неверный тип (не UUID)     | `400 Bad Request` — некорректный формат ID             |

**3. Получение всех объявлений по ID продавца (GET /api/1/{sellerID}/item)**

| **Сценарий**               | **Ожидаемый результат**                                                          |
| -------------------------- | -------------------------------------------------------------------------------- |
| Действительный ID продавца | `200 OK` — возвращается список объявлений, принадлежащих продавцу с указанным ID |
| Неверный тип (не UUID)     | `400 Bad Request` — некорректный формат ID продавца                              |

**4. Получение статистики объявления по ID (GET /api/1/item/{id}/statistics)**

| **Сценарий**                  | **Ожидаемый результат** |
| ----------------------------- | -------------------------------------------------------------------------- | ----------------------- |
| Действительный ID объявления  | `200 OK` — возвращается статистика объявления (likes, viewCount, contacts) |
| ID несуществующего объявления | `404 Not Found` — объявление с указанным ID не найдено                     |
